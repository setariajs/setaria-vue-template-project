# setaria-vue-template-project

业务模块模块前端工程
## 开发环境搭建

[文档](https://github.com/setariajs/setaria/blob/master/docs/DEVELOP_ENVIRONMENT_GUIDE.md)

## 快速开始

```bash
# 取得工程所需第三方依赖
yarn
# 取得最新版本前端框架和公共代码
yarn upgrade:all
# 启动本地开发服务器
yarn dev
```
